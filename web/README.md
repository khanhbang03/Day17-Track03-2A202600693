# MemoryOS showcase

Interactive, dependency-free presentation website for the AI Agent Memory lab.

From the repository root:

```powershell
python -m http.server 8080
```

Then open:

```text
http://localhost:8080/web/
```

Highlights:

- responsive product landing page
- Standard / Long-context benchmark toggle
- complete 8-column benchmark table for both suites
- animated prompt-growth visualization
- comprehensive Baseline vs Advanced comparison cockpit
- correction-aware `User.md` walkthrough
- architecture explorer
- evaluator mode with criterion-by-criterion rubric evidence
- interactive 10-question project analysis and Q&A
- fullscreen presentation mode with arrow-key navigation
- fully usable offline except for the optional Google Fonts request
